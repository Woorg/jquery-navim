navim - test page

navim allows web developers to easily ad vim-style keyboard navigation to their page elements.

right now it's in early development, and only allows for up/down motion. But stay tuned on gfxmonk.net for further announcements.

Currently supported keys:

j, k
down, up
return
perform an action on the currently selected element (by default, this will click the first link in the element)
G, gg
bottom of page, top of page

Planned:

h,l
left and right
n,p
load next and previous pages

The important parts of this page source are the javascript to set the navigation items. In this case, I'm using all pre and li elements - you can use any jQuery collection here though:

Javascript:

function init_vim() {
  $(function() { $("pre,li").vimNavigation(); });
}

$(init_vim);

And of course the CSS. To set a style specific to the currently selected navigation element, add a CSS rule for elements with class navim_active.

If you like, you can also override the "action" callback, like so:

$.vimNavigationAction( function(elem) {
  alert("The current element's contents are: " + elem.html());
});
