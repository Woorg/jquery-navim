navim - test page

navim allows web developers to easily ad vim-style keyboard navigation to their page elements.

right now it's in early development, and only allows for up/down motion. But stay tuned on gfxmonk.net for further announcements.

The important parts of this page source are:

Javascript:

function init_vim() {
	$(function() { $("li").setVimNavigation("vertical"); });
	$(function() { $("ul").setVimNavigation("horizontal"); });
}

$(init_vim);

CSS:

li{
	border:2px solid #ddd;
}

li.navim_active {
	border-color:#422;
}
